         <div class="facet remarks">
            <div class="label">Remarks</div>
            <div class="statement remarks">
               <p>The <span class="att">string</span>, <span class="att">fret</span>, and <span class="att">fing</span> attributes are provided in
                  order to create displayable chord tablature grids. The <span class="att">inth</span> (harmonic interval)
                  attribute may be used to facilitate automated performance of a chord. It gives the
                  number of
                  1/2 steps above the bass. Of course, for the bass note itself, <span class="att">inth</span> should be set
                  to '0'.
               </p>
