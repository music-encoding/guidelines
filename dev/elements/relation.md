         <td class="wovenodd-col1"><strong>Remarks</strong></td>
         <td class="wovenodd-col2">
            <p>The <span class="att">plist</span> and <span class="att">target</span> attributes identify the participants in a
               relationship, while the <span class="att">rel</span> attribute describes the nature of their relationship.
               A mutual relationship can be described using only the <span class="att">plist</span> attribute â€“ the
               <span class="att">target</span> attribute is not necessary. In a non-mutual relationship,
               <span class="att">plist</span> identifies the entities pointed "from", while <span class="att">target</span> specifies
               the entities pointed "to". If the <span class="att">target</span> attribute is present, but the
               <span class="att">plist</span> is not, the relationship is presumed to exist between the parent of the
               current relation element and the entities identified by <span class="att">target</span>.
            </p>
